# GEM MOTORS - Cars for Sale

## Project Overview
GEM MOTORS is a web application designed to showcase a selection of cars for sale. The application provides an elegant and user-friendly interface for users to browse through an inventory of vehicles, complete with images and descriptions. It is built using HTML, CSS, and Tailwind CSS for styling.

## Installation
To get started with the GEM MOTORS project, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/gem-motors.git
   cd gem-motors
   ```
   
2. **Open the Project in a Browser**
   - Simply open the `index.html` file in your preferred web browser.

## Usage
Once the project is running, users can navigate through different sections of the site:
- **Home**: Introductory section to find your dream car.
- **Inventory**: A display of available cars with details including images, descriptions, and prices.
- **Contact**: Information on how to get in touch with GEM MOTORS.

## Features
- Responsive design for optimal viewing on various devices.
- Attractive and modern user interface built with Tailwind CSS.
- Showcases a range of vehicles with images and detailed information.
- Easy navigation between different sections of the website.
- Contact information prominently displayed for customer inquiries.

## Dependencies
The project utilizes the following external libraries:
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs.
- **Font Awesome**: Font library for scalable vector icons.

These libraries are included via CDN links in the `index.html` file.
